El presente proyecto tiene como objetivo el desarrollo de una aplicación móvil orientada a estudiantes, cuyo propósito es mejorar la organización académica y optimizar los procesos de estudio mediante el uso de herramientas digitales e inteligencia artificial.
La aplicación permitirá gestionar tareas, exámenes y horarios de forma centralizada, así como utilizar funcionalidades inteligentes como el resumen automático de contenidos y la generación de preguntas de estudio.
En un contexto en el que el uso del smartphone se ha convertido en el principal medio de acceso a la información, esta solución busca ofrecer una herramienta accesible, eficiente y adaptada al día a día del estudiante.


1. Abstracto
En la actualidad, el uso de dispositivos móviles se ha convertido en una herramienta fundamental en la vida diaria de los estudiantes, especialmente en el ámbito académico. Sin embargo, a pesar de la gran cantidad de aplicaciones disponibles, muchas de ellas no ofrecen una solución completa que permita gestionar de forma eficiente tareas, exámenes y materiales de estudio en un único entorno.
En este contexto, el presente proyecto tiene como objetivo el desarrollo de SmartStudy, una aplicación móvil orientada a estudiantes que permite mejorar la organización académica y optimizar los procesos de estudio mediante el uso de herramientas digitales e inteligencia artificial.
La aplicación integrará funcionalidades como la gestión de tareas, planificación de exámenes, calendario académico y almacenamiento de apuntes. Además, incorporará un sistema de inteligencia artificial capaz de resumir textos y facilitar el aprendizaje, permitiendo al usuario estudiar de forma más eficiente.
Este proyecto surge como respuesta a la necesidad de disponer de una herramienta accesible, centralizada y adaptada al día a día del estudiante, que contribuya a mejorar su rendimiento académico y organización personal. A través del desarrollo de SmartStudy, se busca aprovechar el potencial de la inteligencia artificial aplicada al ámbito educativo, ofreciendo una solución práctica, intuitiva y de utilidad real.

2.1 Introducción
Hasta el momento, el desarrollo de software se ha abordado desde un punto de vista técnico. Sin embargo, en este proyecto se adopta un enfoque orientado al análisis del mercado y a la identificación de necesidades reales de los usuarios.
Antes de desarrollar la aplicación SmartStudy, se realiza un estudio del entorno actual de aplicaciones educativas y de productividad dirigidas a estudiantes, con el objetivo de identificar oportunidades de mejora y detectar carencias existentes.
Este análisis permite asegurar que el proyecto no solo sea técnicamente viable, sino que también aporte valor real a los usuarios, respondiendo a necesidades concretas relacionadas con la organización académica y el estudio.

2.2 Identificación de empresas representativas
Para el desarrollo de SmartStudy, se ha realizado un análisis de aplicaciones existentes en el mercado que ofrecen funcionalidades similares. Entre las más relevantes se encuentran:
Notion: herramienta de organización que permite gestionar tareas y notas, aunque no está enfocada específicamente al ámbito educativo.


Todoist: aplicación de gestión de tareas con sistema de prioridades, pero sin funcionalidades académicas específicas.


MyStudyLife: aplicación orientada a estudiantes que permite gestionar horarios y exámenes, aunque con funcionalidades limitadas en inteligencia artificial.


Google Calendar: ampliamente utilizado para la planificación, pero sin integración directa con tareas académicas ni IA.


Tras analizar estas aplicaciones, se observa que ninguna integra en un único entorno:
Gestión de tareas


Organización académica


Inteligencia artificial aplicada al estudio


Por lo tanto, se detecta una oportunidad clara de mercado para una solución como SmartStudy.


3.1 Identificación de necesidades
Tras el análisis del entorno educativo actual, se identifican las siguientes necesidades en los estudiantes:
1. Organización académica centralizada
 Los estudiantes utilizan múltiples herramientas (agenda, apps, apuntes físicos), lo que dificulta la organización.
2. Gestión eficiente de tareas y exámenes
 Existe dificultad para controlar fechas de entrega y prioridades.
3. Optimización del tiempo de estudio
 Muchos estudiantes invierten demasiado tiempo en resumir contenidos manualmente.
4. Acceso rápido a la información
 Los apuntes suelen estar desorganizados o dispersos.
5. Uso de herramientas inteligentes
 Existe una creciente necesidad de integrar inteligencia artificial en el estudio.






3.2 Planteamiento de soluciones
Para dar respuesta a las necesidades detectadas, se propone el desarrollo de la aplicación SmartStudy, que incluye las siguientes funcionalidades:
Sistema de login seguro para cada usuario


Gestión de tareas (crear, editar, eliminar, marcar como completadas)


Sistema de prioridades en tareas


Calendario académico integrado


Registro de exámenes


Almacenamiento de apuntes


Sistema de inteligencia artificial para:


Resumir textos


Facilitar el estudio


Estas soluciones permiten centralizar la organización académica en una única aplicación, mejorando la eficiencia y productividad del estudiante.


3.4 Aspectos innovadores
El principal elemento innovador de SmartStudy es la integración de inteligencia artificial en una aplicación de organización académica.
A diferencia de otras aplicaciones, SmartStudy no solo permite gestionar tareas, sino que también ayuda activamente en el proceso de estudio mediante:
Generación automática de resúmenes


Apoyo al aprendizaje mediante IA


Integración total entre organización y estudio


Además, la aplicación se caracteriza por:
Simplicidad de uso


Enfoque específico en estudiantes


Centralización de funcionalidades



3.5 Estudio de viabilidad técnica
El desarrollo de la aplicación SmartStudy es técnicamente viable con los conocimientos adquiridos durante el ciclo formativo de Desarrollo de Aplicaciones Multiplataforma.
La aplicación se plantea como un sistema cliente-servidor, en el que el usuario interactúa desde un dispositivo móvil mientras que los datos se gestionan a través de un backend.
Tecnologías propuestas:
Frontend (app móvil):


Android (Java/Kotlin) o Flutter


Backend:


API REST (Spring Boot / Node.js)


Base de datos:


MySQL o Firebase


Autenticación:


Sistema de login con usuario y contraseña


IA:


Integración de API de inteligencia artificial para resumen de textos


Funcionalidades técnicas clave:
CRUD completo de tareas


Gestión de usuarios (login/registro)


Sistema de calendario


Integración con servicios externos (IA)


Conclusión de viabilidad:
El proyecto es completamente viable a nivel técnico, ya que combina funcionalidades ya conocidas (CRUD, login, API) con una integración sencilla de inteligencia artificial mediante servicios externos.
