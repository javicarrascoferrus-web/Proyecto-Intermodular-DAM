FUNCIONES MÍNIMAS 
El presente proyecto tiene como objetivo el desarrollo de una aplicación móvil orientada a estudiantes, cuyo propósito es mejorar la organización académica y optimizar los procesos de estudio mediante el uso de herramientas digitales e inteligencia artificial.
La aplicación permitirá gestionar tareas, exámenes y horarios de forma centralizada, así como utilizar funcionalidades inteligentes como el resumen automático de contenidos y la generación de preguntas de estudio.
En un contexto en el que el uso del smartphone se ha convertido en el principal medio de acceso a la información, esta solución busca ofrecer una herramienta accesible, eficiente y adaptada al día a día del estudiante.


1. Abstracto
Hoy en día los estudiantes usan mucho el móvil para estudiar, pero muchas veces tienen las cosas desorganizadas porque usan varias aplicaciones diferentes para tareas, exámenes y apuntes.
Por eso, en este proyecto voy a desarrollar una aplicación llamada SmartStudy, que sirve para ayudar a los estudiantes a organizarse mejor. En esta app se podrán guardar tareas, ver exámenes en un calendario y guardar apuntes.
Además, la aplicación tendrá una parte de inteligencia artificial que permitirá resumir textos, lo que ayudará a estudiar más rápido.
El objetivo de este proyecto es crear una herramienta útil, sencilla y que realmente ayude a los estudiantes en su día a día.



2.1 INTRODUCCIÓN (BÚSQUEDA DE INFORMACIÓN)


2.1 Introducción
Antes de empezar a programar la aplicación, es importante analizar qué necesitan los usuarios y qué aplicaciones existen actualmente.
En este caso, se ha analizado el uso de aplicaciones de organización y estudio para entender qué problemas tienen los estudiantes y cómo se puede mejorar con SmartStudy.

2.2 Identificación de empresas representativas
Para el desarrollo de SmartStudy, se ha realizado un análisis de aplicaciones existentes en el mercado que ofrecen funcionalidades similares. Entre las más relevantes se encuentran:
Notion: herramienta de organización que permite gestionar tareas y notas, aunque no está enfocada específicamente al ámbito educativo.


Todoist: aplicación de gestión de tareas con sistema de prioridades, pero sin funcionalidades académicas específicas.


MyStudyLife: aplicación orientada a estudiantes que permite gestionar horarios y exámenes, aunque con funcionalidades limitadas en inteligencia artificial.


Google Calendar: ampliamente utilizado para la planificación, pero sin integración directa con tareas académicas ni IA.


Tras analizar estas aplicaciones, se observa que ninguna integra en un único entorno:
Gestión de tareas


Organización académica


Inteligencia artificial aplicada al estudio


Por lo tanto, se detecta una oportunidad clara de mercado para una solución como SmartStudy.




3.1 Identificación de necesidades

3.5 Estudio de viabilidad técnica
El desarrollo de la aplicación SmartStudy es técnicamente viable con los conocimientos adquiridos durante el ciclo formativo de Desarrollo de Aplicaciones Multiplataforma.
La aplicación se plantea como un sistema cliente-servidor, en el que el usuario interactúa desde un dispositivo móvil mientras que los datos se gestionan a través de un backend.
Tecnologías propuestas:
App móvil (Android o Flutter)
Backend con API
Base de datos
IA mediante API externa

Conclusión de viabilidad:
El proyecto es completamente viable a nivel técnico, ya que combina funcionalidades ya conocidas (CRUD, login, API) con una integración sencilla de inteligencia artificial mediante servicios externos.
